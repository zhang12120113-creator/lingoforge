## ✨ 产品定位

LingoForge 不是单纯的背单词工具,也不是单纯的阅读器。我们将英语学习拆分为 **六层闭环结构**,让用户在真实语境中输入、在系统训练中记忆、在智能反馈中提效。

> **当前版本**:打字背词 · 分级阅读 · 视频语料 三大输入与记忆模块已上线
> **后续模块**:训练中心 · 听力强化 · 语法 · AI 教练 · 写作批改

---

## 🎯 六层学习闭环

| 层级 | 作用 | 当前状态 |
|:---|:---|:---|
| **输入层** | 分级文章、视频字幕、真实语料,接触"活的英语" | ✅ **阅读 + 语料中心已上线** |
| **规则层** | 语法体系树、长难句拆解、AI 薄弱点讲解 | 🚧 语法模块规划中 |
| **小记忆层** | 单词卡片、拼写训练、错题本、个人词本、遗忘曲线 | ✅ **打字背词 + 三类词本已上线** |
| **训练层** | 跟读模仿、听写练习、完形填空、中英互译 | 🚧 训练中心开发中 |
| **应用层** | 邮件写作、段落写作、口语对话、翻译练习 | 🚧 AI 教练开发中 |
| **反馈层** | AI 纠错、错因分析、学习报告、能力成长曲线 | 🚧 开发中 |

## 🚀 已上线功能(MVP)

### 📝 打字背单词
- **20 本精准词库**:中高考、CET-4/6、考研、雅思、托福、SAT、专四专八,以及 **6 类专业英语**(程序员、航海、商务、汽修、厨师、电工)
- **科学分章记忆**:每章 25 词,学习负担可控,章节导航可展开/收起
- **沉浸式打字练习**:边打边记,肌肉记忆 + 视觉记忆双通道
- **下一词预览**:右上角实时预览,提前准备拼写,流程更顺畅
- **错词自动收录**:拼写错误自动进入错题本,针对性强化
- **章节单词直达**:章节列表中点击任意单词,直接定位到该词

### 📖 阅读模块
- 分级文章列表 + 沉浸式文章详情阅读
- 点击查词,即点即查,不打断阅读节奏
- 生词一键收藏至 **阅读词本**,可在打字模块中专项练习

### 🎬 语料中心
- 视频 + 字幕双视图,在真实语境中学英语
- 字幕模式切换(中英 / 英文 / 隐藏)、句子级循环、当前句聚焦
- **键盘快捷键**:`空格` 暂停 / `←→` 前后跳转 5 秒 / `↑↓` 切换字幕 / `L` 切换循环
- 字幕点词即查,收藏至 **语料词本**,从看视频沉淀出专属词表

### 👤 个人中心
- 学习时长分模块统计:打字 / 阅读 / 语料 / 训练独立计时
- 三类个人词本:**错题本** · **阅读词本** · **语料词本**,全部可作为词库进入打字训练
- 深色 / 浅色双主题,移动端完整适配

## 📚 词库数据

通用词库来源于开源项目 [qwerty-learner](https://github.com/RealKai42/qwerty-learner) 及公开考试大纲,经以下处理:
- 清洗去重,保证词库纯净
- 按 **25 词/章** 强制切分,控制单次学习负荷
- **专业英语**(航海 / 商务 / 汽修 / 厨师 / 电工)词库由项目自建,基于行业标准用语和官方词表整理
