## ✨ 产品定位

LingoForge 不是单纯的背单词工具，也不是单纯的阅读器。我们将英语学习拆分为 **六层闭环结构**，让用户在真实语境中输入、在系统训练中记忆、在智能反馈中提效。

> **当前版本**：打字记忆模块已上线 | **后续模块**：阅读 · 语法 · AI 教练 · 写作

---

## 🎯 六层学习闭环

| 层级 | 作用 | 当前状态 |
|:---|:---|:---|
| **输入层** | 分级文章、热点文章、真实语料,接触"活的英语" | 🚧 阅读模块开发中 |
| **规则层** | 语法体系树、长难句拆解、AI 薄弱点讲解 | 🚧 语法模块开发中 |
| **小记忆层** | 单词卡片、拼写训练、发音训练、错词本、遗忘曲线复习 | ✅ **打字背单词已上线** |
| **训练层** | 打字背词、跟读模仿、听写练习、完形填空、中英互译 | ✅ **打字训练已上线** |
| **应用层** | 邮件写作、段落写作、口语对话、翻译练习 | 🚧 AI 教练开发中 |
| **反馈层** | AI 纠错、错因分析、学习报告、能力成长曲线 | 🚧 开发中 |

---

## 🚀 已上线功能（MVP）

### 打字背单词模块
- **15 个精准词库**：覆盖中高考、CET-4/6、考研、雅思、托福、SAT、程序员英语
- **科学分章记忆**：每章 25 词，学习负担可控，章节导航可展开/收起
- **沉浸式打字练习**：边打边记，肌肉记忆 + 视觉记忆双通道
- **下一词预览**：右上角实时预览，提前准备拼写，流程更顺畅
- **深色 / 浅色双主题**：护眼模式适配长时间学习
- **点击跳转**：章节列表中点击任意单词,直接定位到该词
- **错词自动收录**：拼写错误自动进入错词本，针对性强化

## 🛠 技术栈

| 层级 | 技术选型 |
|:---|:---|
| 前端 | React 18 + Vite + Tailwind CSS |
| 状态管理 | React Hooks + Context |
| 部署 | Vercel（全球 CDN，自动构建） |
| 数据 | JSON 静态词库（15 本词书，已清洗去重） |
| 后续后端 | Node.js / FastAPI + MySQL + Redis（用户系统 & 进度同步） |

## 📚 词库数据

词库数据来源于开源项目 [qwerty-learner](https://github.com/RealKai42/qwerty-learner) 及公开考试大纲，经以下处理：
- 清洗去重，保证词库纯净
- 按 **25 词/章** 强制切分，控制单次学习负荷
- 补充 CET-4/6 及程序员词库完整性校验
