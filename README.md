# LingoForge

> 面向中国学习者的沉浸式英语打字记忆平台 —— 在打字中背单词，在背诵中练打字。

[🌐 线上体验](https://www.lingoforge.fun/) · [🐛 问题反馈](https://github.com/zhang12120113-creator/lingoforge/issues)

---

## ✨ 功能亮点

- **15 个精准词库**：覆盖中高考、CET-4/6、考研、雅思、托福、SAT、程序员英语
- **科学分章记忆**：每章 25 词，学习负担可控
- **沉浸式打字练习**：边打边记，肌肉记忆 + 视觉记忆双通道
- **深色 / 浅色双主题**：护眼模式适配长时间学习
- **章节导航**：左侧可展开章节列表，点击任意单词直接跳转
- **下一词预览**：右上角实时预览，提前准备拼写

## 🛠 技术栈

- React 18 + Vite
- Tailwind CSS
- 部署：Vercel（全球 CDN）

## 🚀 本地运行

```bash
git clone https://github.com/zhang12120113-creator/lingoforge.git
cd lingoforge
npm install
npm run dev
```

## 📚 词库数据

词库数据来源于开源项目 qwerty-learner 及公开考试大纲，经清洗、去重、按 25 词/章切分。
